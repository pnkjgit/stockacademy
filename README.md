# StockAcademy

StockAcademy is a learning platform where users can learn about the **Indian Stock Market from scratch**.

## Tech Stack

Frontend:

- React
- TypeScript
- Vite

Backend:

- ASP.NET Core Web API
- .NET

## Project Structure

stockacademy
├── frontend (React App)
├── backend (ASP.NET API)

## Running the Project

### Run Backend

cd backend
dotnet run

Backend runs at:
http://localhost:5043

### Run Frontend

cd frontend
npm install
npm run dev

Frontend runs at:
http://localhost:5173
